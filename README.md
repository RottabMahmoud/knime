# KNIME Case Study Description

## What the Application does

- Hello Knime

## Technologies

- Vue
- Vue Router
- Vuex (Installed but not used)
- JavaScript
- Vuetify

## Project Installation

```bash
git clone https://github.com/Rottabx/knime.git
cd knime
yarn
```

## To Start the App

```bash
yarn serve
```

## For Building

```bash
yarn build
```

## Usage

Used Vuetify as a Component Library.
